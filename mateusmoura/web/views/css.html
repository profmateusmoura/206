3. Posicionamento no CSS

Static (Padrão)

O posicionamento static é o comportamento padrão de todos os elementos. Eles são renderizados no fluxo normal do documento, um após o outro.

/* Exemplo de static (padrão) */
.box {
    position: static;
}
Static

Relative

O position: relative permite que o elemento seja movido em relação à sua posição original sem sair do fluxo do documento. Outros elementos ainda se comportam como se ele estivesse em sua posição original.

/* Exemplo de position relative */
.relative-box {
    position: relative;
    top: 20px;
    left: 50px;
}
Relative

Absolute

O position: absolute remove o elemento do fluxo normal e o posiciona em relação ao seu contêiner pai mais próximo que tenha um posicionamento definido (como relative, absolute ou fixed).

/* Exemplo de position absolute */
.absolute-box {
    position: absolute;
    top: 20px;
    left: 50px;
}
Absolute

Fixed

O position: fixed posiciona o elemento em relação à janela de visualização (viewport) e o mantém fixo quando a página é rolada.

/* Exemplo de position fixed */
.fixed-box {
    position: fixed;
    bottom: 20px;
    right: 20px;
}
Fixed

Sticky

O position: sticky combina comportamentos de relative e fixed. O elemento se comporta como relative até que atinja uma posição específica na tela, momento em que ele "gruda" e passa a se comportar como fixed.

/* Exemplo de position sticky */
.sticky-box {
    position: -webkit-sticky; /* Para navegadores WebKit */
    position: sticky;
    top: 10px;
}
Sticky

4. Transições e Animações

Transições

/* Transição de cor */
button {
    background-color: #3498db;
    color: white;
    padding: 10px 20px;
    border: none;
    transition: background-color 0.5s ease;
}

button:hover {
    background-color: #2ecc71;
}

Animações

/* Animação de rotação */
@keyframes rotacao {
    from {
        transform: rotate(0deg);
    }
    to {
        transform: rotate(360deg);
    }

.rotating-box {
    width: 100px;
    height: 100px;
    background-color: #3498db;
    animation: rotacao 2s infinite linear;
}

5. Outras Propriedades Comuns

Exemplo de Layout com Flexbox e Alinhamento de Texto

/* Exemplo de uso do display, justify-content e align-items */
.container {
    display: flex;
    justify-content: center; /* Centraliza horizontalmente */
    align-items: center;    /* Centraliza verticalmente */
    height: 200px;          /* Define a altura do container */
    background-color: #f1f1f1;
}

/* Exemplo de alinhamento de texto */
.texto {
    text-align: center;
    font-size: 18px;
    color: #333;
}

5. Outras Propriedades Comuns

Layout com Flexbox

/* Exemplo de layout com Flexbox */
.container-flex {
    display: flex;
    justify-content: center; /* Centraliza os itens horizontalmente */
    align-items: center;     /* Centraliza os itens verticalmente */
    height: 200px;           /* Define a altura do container */
    background-color: #f1f1f1; /* Cor de fundo do container */
}

Neste exemplo, utilizamos o Flexbox para centralizar os elementos tanto na horizontal quanto na vertical. A propriedade display: flex transforma o container em um flex container; justify-content alinha os itens no eixo principal, e align-items ajusta o alinhamento no eixo cruzado.

Alinhamento de Texto

/* Exemplo de alinhamento de texto */
.titulo {
    text-align: center; /* Centraliza o texto */
    font-size: 24px;
    color: #555;
}

A propriedade text-align controla o alinhamento horizontal do texto dentro de um elemento. Neste exemplo, o texto é centralizado.

Uso Avançado do Display com Grid

/* Exemplo com display: grid */
.container-grid {
    display: grid;
    grid-template-columns: repeat(3, 1fr); /* Cria três colunas iguais */
    gap: 10px; /* Espaçamento entre os itens */
    background-color: #e0e0e0; /* Cor de fundo do container */
    padding: 10px;
}

O display: grid permite criar layouts complexos com pouco código. Aqui, definimos um grid de três colunas iguais usando a função repeat e 1fr para repartir igualmente o espaço.

Layout Responsivo com Flexbox

/* Exemplo combinando propriedades para um layout responsivo */
.container-responsive {
    display: flex;
    flex-wrap: wrap;               /* Permite que os itens quebrem para a próxima linha */
    justify-content: space-around; /* Distribui o espaço igualmente entre os itens */
    align-items: center;           /* Centraliza itens verticalmente */
    height: auto;                  /* Altura ajustável conforme o conteúdo */
    background-color: #fafafa;
    padding: 20px;
}

.item {
    flex: 1 1 200px;   /* Itens flexíveis com tamanho base de 200px */
    margin: 10px;
    background-color: #ddd;
    padding: 20px;
    text-align: center;
}

Este exemplo mostra como combinar as propriedades display: flex, flex-wrap, justify-content e align-items para criar um layout responsivo que se adapta ao tamanho da tela. A propriedade flex: 1 1 200px define a capacidade de crescimento e encolhimento dos itens.

Exemplos Complementares

/* Utilizando propriedades de altura, display e background-color */
.box-destaque {
    display: block;          /* Exibe o elemento como bloco */
    height: 150px;           /* Define altura fixa */
    background-color: #ffeb3b; /* Define uma cor de fundo de destaque */
    line-height: 150px;      /* Alinha verticalmente o texto */
    text-align: center;      /* Centraliza o texto horizontalmente */
}

Nesta última demonstração, combinamos as propriedades display, height, background-color e text-align para criar um bloco de destaque com um conteúdo centralizado. Essa abordagem pode ser útil para criar banners, chamadas de atenção ou seções especiais na página.
